# @addfox/rsbuild-plugin-extension-monitor

[中文](README-zh_CN.md) | English

---

Dev-only Rsbuild plugin that injects error monitoring into entry scripts and emits a monitor UI under `_addfox-monitor/`. Injected by the CLI pipeline; users do not configure it directly.
