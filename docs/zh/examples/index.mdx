## 运行方式

在 addfox 仓库根目录执行安装后，进入任意示例目录：

<Tabs>
  <Tab label={<TabLabelPnpm />}>

```bash
pnpm install
cd examples/addfox-with-vue   # 或其他示例
pnpm dev                   # 开发模式
pnpm build                 # 构建
```

  </Tab>
  <Tab label={<TabLabelNpm />}>

```bash
npm install
cd examples/addfox-with-vue   # 或其他示例
npm run dev                # 开发模式
npm run build              # 构建
```

  </Tab>
  <Tab label={<TabLabelYarn />}>

```bash
yarn install
cd examples/addfox-with-vue   # 或其他示例
yarn dev                   # 开发模式
yarn build                 # 构建
```

  </Tab>
  <Tab label={<TabLabelBun />}>

```bash
bun install
cd examples/addfox-with-vue   # 或其他示例
bun run dev                # 开发模式
bun run build              # 构建
```

  </Tab>
</Tabs>

构建产物在 `.addfox/extension`（或该示例配置的 outputRoot/outDir），在浏览器中加载该目录即可；使用 dev 时框架可自动打开浏览器并加载扩展。
