# React

Addfox soporta completamente React, puedes usar JSX/TSX para desarrollar todas las entradas de la extensiГіn.

## InstalaciГіn

Seleccionar plantilla React al crear el proyecto:


## ConfiguraciГіn

```ts

  plugins: [pluginReact()],
});
```

## CГіdigo de ejemplo

### Popup

```tsx
// app/popup/index.tsx
import { createRoot } from 'react-dom/client';
import { App } from './App';
import './index.css';

const container = document.getElementById('root');
const root = createRoot(container!);
root.render(<App />);
```

```tsx
// app/popup/App.tsx
import { useState } from 'react';

export function App() {
  const [count, setCount] = useState(0);

  return (
    <div className="p-4">
      <h1 className="text-xl font-bold">Hello React!</h1>
      <button 
        onClick={() => setCount(c => c + 1)}
        className="mt-2 px-4 py-2 bg-blue-500 text-white rounded"
      >
        Count: {count}
      </button>
    </div>
  );
}
```

Si **omites** el siguiente `index.html` y solo conservas `index.tsx`, la construcciГіn generarГЎ automГЎticamente HTML, que contiene **`id="root"`**, **title** y **favicon** sincronizados con **`manifest.name`** / **`manifest.icons`** (ver [Entradas basadas en archivos](/es/guide/entry/file-based)). El siguiente ejemplo es una plantilla personalizada **opcional**:

```html
<!-- app/popup/index.html (opcional; al personalizar necesitas escribir title / icono tГє mismo) -->
<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8" />
    <title>Popup</title>
  </head>
  <body>
    <div id="root"></div>
    <script type="module" src="./index.tsx" data-addfox-entry></script>
  </body>
</html>
```

### Content Script

```tsx

root.render(<ContentApp />);
```

## GestiГіn de estado

Puedes usar cualquier soluciГіn de gestiГіn de estado de React:

### Zustand

```bash
pnpm add zustand
```

```ts
// app/store.ts
import { create } from 'zustand';

interface Store {
  count: number;
  increment: () => void;
}

export const useStore = create<Store>((set) => ({
  count: 0,
  increment: () => set((state) => ({ count: state.count + 1 })),
}));
```

### Jotai

```bash
pnpm add jotai
```


## InteracciГіn con la API de Chrome

```tsx
import { useEffect, useState } from 'react';

export function TabList() {
  const [tabs, setTabs] = useState<chrome.tabs.Tab[]>([]);

  useEffect(() => {
    chrome.tabs.query({}, setTabs);
  }, []);

  const activateTab = (tabId: number) => {
    chrome.tabs.update(tabId, { active: true });
  };

  return (
    <ul>
      {tabs.map((tab) => (
        <li key={tab.id} onClick={() => activateTab(tab.id!)}>
          {tab.title}
        </li>
      ))}
    </ul>
  );
}
```
