# Entry

This is the overview page for the Addfox entry system.
Entries are recommended to be understood in two categories:

- **File-based**: Auto-discover entries by directory convention
- **Config-based**: Explicit configuration via `entry + manifest` in `addfox.config.ts`

If you are new to entry configuration, we recommend reading in this order:

- [Entry Concepts](/guide/entry/concept)
- [File-based Entry](/guide/entry/file-based)
- [Config-based Entry](/guide/entry/config-based)
