import { Tab, Tabs, TabLabelPnpm, TabLabelNpm, TabLabelYarn, TabLabelBun } from '@rspress/core/theme';

# Sass

## Установка

Необходимо установить плагин `@rsbuild/plugin-sass`:

<Tabs>
  <Tab label={<TabLabelPnpm />}>

```bash
pnpm add -D @rsbuild/plugin-sass sass
```

  </Tab>
  <Tab label={<TabLabelNpm />}>

```bash
npm install -D @rsbuild/plugin-sass sass
```

  </Tab>
  <Tab label={<TabLabelYarn />}>

```bash
yarn add -D @rsbuild/plugin-sass sass
```

  </Tab>
  <Tab label={<TabLabelBun />}>

```bash
bun add -D @rsbuild/plugin-sass sass
```

  </Tab>
</Tabs>

## Конфигурация

Добавьте плагин в `addfox.config.ts`:

```ts
import { defineConfig } from "addfox";
import { pluginSass } from "@rsbuild/plugin-sass";

export default defineConfig({
  plugins: [pluginSass()],
});
```

## Использование

После установки и настройки можно напрямую создавать и импортировать файлы `.scss`:

```scss
/* app/popup/index.scss */
$brand: #7c3aed;

.card {
  border: 1px solid $brand;
  color: $brand;
}
```

```ts
import "./index.scss";
```

## Примечания

- При использовании модульных стилей можно комбинировать с `*.module.scss`
- Плагин автоматически обрабатывает компиляцию и импорт файлов Sass
