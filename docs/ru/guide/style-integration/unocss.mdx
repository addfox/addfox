
  </Tab>

## Конфигурация

Создайте `postcss.config.mjs`:

```js
export default {
  plugins: {
    "@unocss/postcss": {},
  },
};
```

Опционально: создайте `uno.config.ts` для расширения правил.

## Использование

Импортируйте UnoCSS в вашем стилевом entry:

```css
/* app/popup/index.css */
@unocss;
```

И импортируйте этот CSS в entry скрипте:

```ts
import "./index.css";
```

Затем можно использовать атомарные классы UnoCSS в компонентах.
