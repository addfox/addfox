import { Tab, Tabs, TabLabelPnpm, TabLabelNpm, TabLabelYarn, TabLabelBun } from '@rspress/core/theme';

# Less

## Установка

Необходимо установить плагин `@rsbuild/plugin-less`:

<Tabs>
  <Tab label={<TabLabelPnpm />}>

```bash
pnpm add -D @rsbuild/plugin-less less
```

  </Tab>
  <Tab label={<TabLabelNpm />}>

```bash
npm install -D @rsbuild/plugin-less less
```

  </Tab>
  <Tab label={<TabLabelYarn />}>

```bash
yarn add -D @rsbuild/plugin-less less
```

  </Tab>
  <Tab label={<TabLabelBun />}>

```bash
bun add -D @rsbuild/plugin-less less
```

  </Tab>
</Tabs>

## Конфигурация

Добавьте плагин в `addfox.config.ts`:

```ts
import { defineConfig } from "addfox";
import { pluginLess } from "@rsbuild/plugin-less";

export default defineConfig({
  plugins: [pluginLess()],
});
```

## Использование

После установки и настройки можно напрямую создавать и импортировать файлы `.less`:

```less
/* app/popup/index.less */
@brand: #2563eb;

.btn {
  background: @brand;
  color: #fff;
}
```

```ts
import "./index.less";
```

## Примечания

- Рекомендуется использовать именованные пространства имен классов в сценариях content для избежания загрязнения стилей страницы
- Плагин автоматически обрабатывает компиляцию и импорт файлов Less
