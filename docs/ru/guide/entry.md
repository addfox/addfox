# Entry

Это обзорная страница системы entry Addfox.  
Entry рекомендуется понимать в двух категориях:

- **На основе файлов**: автообнаружение entry по соглашениям о директориях;
- **На основе конфигурации**: явная конфигурация через `entry + manifest` в `addfox.config.ts`.

Если вы впервые сталкиваетесь с конфигурацией entry, рекомендуется читать в следующем порядке:

- [Концепция entry](/ru/guide/entry/concept)
- [Entry на основе файлов](/ru/guide/entry/file-based)
- [Entry на основе конфигурации](/ru/guide/entry/config-based)
