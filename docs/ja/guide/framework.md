# フレームワークサポート

Addfox は主要なフロントエンドフレームワークをサポートし、対応するサンプルとベストプラクティスを提供します。  
使用している技術スタックに応じて、対応するドキュメントを参照してください：

- [React](/ja/guide/framework/react)
- [Vue](/ja/guide/framework/vue)
- [Svelte](/ja/guide/framework/svelte)
- [Solid](/ja/guide/framework/solid)
- [Preact](/ja/guide/framework/preact)
