# エントリー

ここは Addfox エントリーシステムの概要ページです。  
エントリーは以下の2つのカテゴリに分けて理解することをお勧めします：

- **ファイルベース**：ディレクトリ規約に従って自動的にエントリーを検出;
- **設定ベース**：`addfox.config.ts` で `entry + manifest` を使用して明示的に設定。

初めてエントリー設定に触れる場合は、以下の順序で読むことをお勧めします：

- [エントリーコンセプト](/ja/guide/entry/concept)
- [ファイルシステムエントリー](/ja/guide/entry/file-based)
- [設定駆動エントリー](/ja/guide/entry/config-based)
