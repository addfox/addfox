# サンプル

このページでは、addfox リポジトリの拡張機能サンプルを一覧表示します。各サンプルは独立したディレクトリで、`package.json` と `addfox.config.ts` を含みます。リポジトリのルートで pnpm / npm / yarn / bun を使用して依存関係をインストールした後、対応するサンプルディレクトリで dev または build を実行して、実行またはビルドできます。

## 実行方法

addfox リポジトリのルートでインストールを実行した後、任意のサンプルディレクトリに移動：

<Tabs>
  <Tab label={<TabLabelPnpm />}>

```bash
pnpm install
cd examples/addfox-with-vue   # または他のサンプル
pnpm dev                   # 開発モード
pnpm build                 # ビルド
```

  </Tab>
  <Tab label={<TabLabelNpm />}>

```bash
npm install
cd examples/addfox-with-vue   # または他のサンプル
npm run dev                # 開発モード
npm run build              # ビルド
```

  </Tab>
  <Tab label={<TabLabelYarn />}>

```bash
yarn install
cd examples/addfox-with-vue   # または他のサンプル
yarn dev                   # 開発モード
yarn build                 # ビルド
```

  </Tab>
  <Tab label={<TabLabelBun />}>

```bash
bun install
cd examples/addfox-with-vue   # または他のサンプル
bun run dev                # 開発モード
bun run build              # ビルド
```

  </Tab>
</Tabs>

ビルド成果物は `.addfox/extension`（または該当サンプルで設定された outputRoot/outDir）にあります。このディレクトリをブラウザで読み込むことができます。dev を使用する場合、フレームワークは自動的にブラウザを開いて拡張機能を読み込みます。
