# UnoCSS

## 설치

<Tabs>
  <Tab label={<TabLabelPnpm />}>

```bash
pnpm add -D unocss @unocss/postcss
```

  </Tab>
  <Tab label={<TabLabelNpm />}>

```bash
npm install -D unocss @unocss/postcss
```

  </Tab>
  <Tab label={<TabLabelYarn />}>

```bash
yarn add -D unocss @unocss/postcss
```

  </Tab>
  <Tab label={<TabLabelBun />}>

```bash
bun add -D unocss @unocss/postcss
```

  </Tab>
</Tabs>

## 구성

`postcss.config.mjs`를 생성합니다:

```js
export default {
  plugins: {
    "@unocss/postcss": {},
  },
};
```

선택 사항: 규칙 확장을 위해 `uno.config.ts`를 생성합니다.

## 사용

Entry 스타일에서 UnoCSS를 가져옵니다:

```css
/* app/popup/index.css */
@unocss;
```

그리고 Entry 스크립트에서 이 CSS를 가져옵니다:

```ts
import "./index.css";
```

그런 다음 컴포넌트에서 UnoCSS 원자 클래스를 사용할 수 있습니다.
